WeatherWarningHK
Hyper-Local Rainfall Time-of-Arrival Warning System
Project Overview

This GIS framework delivers hyper-local rainfall time-of-arrival (TOA) predictions for every government Wi-Fi hotspot in Hong Kong. By integrating the HKO Gridded Rainfall Nowcast dataset with the CSDI Wi-Fi.HK network (4,400+ locations), citizens receive point-specific warnings — not just regional alerts.

Methodology — v3 Ray-Cast Engine
01
4×4 Wide-Area Tile Grid — Fetches a 4×4 tile mosaic at zoom 7 centred on HK (≈1158 km × 1158 km). Covers entire typhoon outer rainbands and large MCS systems that exceed 500 km across — the old 2×2 grid only covered 580 km.
02
Quality-Weighted Block Matching — TREC-style SSD block matching runs across the full 1024 px composite. Blocks weighted by inverse-SSD (better matches count more). Up to three frame pairs averaged for temporal stability. Separate thresholds: ECHO_TRACK = 25 for motion tracking, ECHO_WARN = 75 (≈ 2 mm/h) for citizen alerts. Light drizzle is tracked for motion but never triggers warnings.
03
Per-Spot Ray-Cast TOA — Every hotspot casts its own ray towards the incoming storm direction (not from a shared centroid). A ±15° fan of three rays handles angular uncertainty in the motion vector; the minimum TOA wins. This is the core innovation — each spot gets an independent, location-specific arrival time.
04
Confirmed Leading Edge — Requires 3 consecutive pixels above ECHO_WARN (alpha > 75, ≈ 2 mm/h) before declaring arrival. Rainfall below 2 mm/h is excluded — this eliminates drizzle and radar noise that would otherwise cause false warnings. TOA distance is back-stepped to the first pixel of the confirmed run.
05
Raining-Now Detection — A 3×3 neighbourhood (9-pixel check) at the spot location. If ≥3 of 9 pixels exceed ECHO_TRACK, the spot is inside the current echo → TOA = 0, class = Warning, hint = "RAINING NOW". Prevents a single background pixel from triggering false positives.
06
Classification & Display — Warning ≤20 min (red) · Caution 21–45 min (amber) · Safe >45 min (green) · Watch (no echo, blue). Spots update every 10 min. Low-confidence motion (fewer than 8 matched blocks) shown with dashed banner.
ArcGIS Online Deployment

This app is ArcGIS Online-ready. To publish as an official web mapping application:

// In <script>, fill in ARCGIS_CONFIG: PORTAL_URL: 'https://[org].maps.arcgis.com' APP_ID: '[OAuth Client ID from app registration]' REDIRECT_URI: '[your published URL]' WEBMAP_ID: '[optional — replaces Lands Dept basemap]' // Then uncomment the IdentityManager block // and add OAuthInfo to require() imports.
1
Register app at arcgis.com → Content → Register App (Web Mapping Application type)
2
Copy Client ID → paste as APP_ID in ARCGIS_CONFIG. Set redirect URI to match.
3
Upload index.html to ArcGIS Online → Content → Add Item → Web Mapping App
4
Share item publicly or to your organization. Embed in Experience Builder as an iframe widget if needed.
